## Цель проекта

Демонстрация инженерных навыков в рамках тестового задания: работа с существующей кодовой базой, разделение ответственности между слоями приложения, управление состоянием (URL, браузер, API), обработка граничных случаев и написание тестов.

## Суть проекта

Приложение представляет собой рабочее место оператора поддержки, где можно:

- Просматривать список заявок
- Открывать карточку конкретной заявки
- Отмечать заявки как проверенные
- Использовать Command Palette (`Ctrl/⌘ K`) для быстрой навигации и действий
- Видеть список последних открытых заявок для быстрого возврата к контексту

Все данные — моковые, backend не требуется.

## Как запустить

### Требования

- Node.js ≥ 18
- npm ≥ 9

### Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка проекта
npm run build

# Запуск тестов
npm test

# Запуск тестов в режиме наблюдения
npm run test:watch
```

Приложение будет доступно по адресу `http://localhost:5173`.
